# LandokmaiWeb
LANDOKMAI WEB — Full-Stack Flower Shop Application

Overview{
Landokmai Web คือเว็บขายดอกไม้แบบ Full-Stack พัฒนาโดยใช้
-Frontend : HTML, CSS, JavaScript 
-Backend : Node.js + Express 
-Database : MySQL 
รองรับทั้ง User และ Admin, มีระบบตะกร้าสินค้า การสั่งซื้อ และระบบจัดการบทความ (Blog)
}
