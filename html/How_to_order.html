How To Order

สั่งสินค้ากับเราง่ายๆ ได้ทุกช่องทาง

A) ผ่าน Website 📍

  1. คุณสามารถเลือกดอกไม้ใส่ตะกร้า
  2. ชำระเงินผ่าน บัตรเครดิต หรือ โอนเงิน
  3. แจ้งการชำระเงินที่
    Email : Landokmai.co@gmail.com

*กรณีชำระผ่านบัตรเครดิตไม่ต้องแจ้งชำระเงินก็ได้ค่ะ ระบบจะจัดการให้อัตโนมัติ

B) ผ่าน Line 💬

  1. เพิ่มเพื่อนใน Line ID : @Landokmai  หรือคลิก : L Add Friends
  2. เลือกแบบดอกไม้ที่ถูกใจ และส่งรูปหรือรหัสของเราให้กับแอดมิน Line ของเรา
  3. ชำระเงินตามยอด และแจ้งการชำระเงินผ่าน Line
หมายเหตุ: หากต้องการใบเสร็จ/ใบกำกับภาษี โปรดแจ้งข้อมูลชื่อบริษัท ที่อยู่ และเลขประจำตัวผู้เสียภาษีกับแอดมิน
